Classical Bits

This is a classical bit, which can exist in a state of either 0 or 1 and nothing else. These states can also be thought of as off/on, true/false, low/high, etc. A bit simply represents a single unit of information as a binary. Once a bit is set to a state it will stay in that state until you do something to change it. This probably seems pretty ordinary because the computers we normally use rely on these classical bits. You can click on the bit to change it from 1 to 0. Once you are ready you can move on to the next page.
